# deloitte-app
Application for Deloitte - Coded by Marco Lavielle

Tech Stack:  
MEAN stack  
MongoDB  
AngularJS  
Express  
NodeJS  

Deployed to Heroku:
https://marco-lavielle-deloitte.herokuapp.com/
